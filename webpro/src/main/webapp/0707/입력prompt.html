확인 버튼 클릭하면 두개의 정수를 입력 받는다.
    입력 받은 두 정수를 더하고 div에 출력한다.

    스크립트에서 html요소를 검색, 접근방법
    document.getElementById("id 이름");
    document.getElementByClassName("class 이름");
    document.getElementByTagName("tag" 이름");

    es6
    document.querySelector('#id이름');
    document.querySelector('.class이름');
    document.querySelector('tag이름');

    document.querySelectorAll('#id이름');
    document.querySelectorAll('.class이름');
    document.querySelectorAll('tag이름');
